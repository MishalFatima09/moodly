**📌 Moodly — Pinterest Replica (Kotlin Android App)**

Moodly is a modern Pinterest-inspired Android application built using Kotlin, designed to provide a smooth, visually engaging experience for discovering, browsing, and saving images in a masonry-style feed.

**Features
**
- Pinterest-style Masonry Grid Feed
- Search pins by keywords
- Save & bookmark favorite posts
- Like/unlike posts
- Responsive Material Design UI
- Smooth image loading & caching
- Dynamic feed updates
-Clean multi-screen navigation


	
🧱 Tech Stack

Language: Kotlin
UI: XML + Material Design
Image Loading: Glide
Async Handling: Coroutines
Backend: PHP (Core PHP scripts) 

 Architecture Overview
📌 Explanation
UI Layer: Displays Pinterest-style feed & handles user interactions
ViewModel: Manages UI state and business logic
Repository: Single source of truth
Data Sources: API or local database
